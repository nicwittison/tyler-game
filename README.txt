Power Run — Lit Electrical Tasmania (v8)
=========================================

New in this version:
- **Mobile controls**: on-screen buttons for Left, Right, Plug, and Reset.
  - Works with multi-touch; hold a direction and tap Plug at the same time.
  - Buttons are only shown on touch devices (or small screens).

Still included:
- HiDPI sharp rendering, overlay fix, proper lamp alignment.
- Hat drawn to match head curve.
- ATO van chases you after the first light (game over if it catches you).

How to run:
1) Unzip.
2) Open `index.html` on desktop or mobile (Chrome, Safari, Firefox, Edge).

Controls:
- Desktop: ← → or A/D, Plug (E), Reset (R)
- Mobile: use the on-screen buttons.
